## song metadata
song metadata is meant to help you create more accurate songdesc's, by giving you the exact information for a song!

### usage:
![metadata usage](https://github.com/devvieram/musix/blob/main/docs/bin/metadata-usage.gif)

  "explicit": false,
  "genres": [
    "Pop"

  "length": {
    "readable": "03:01",
